## Icon

PiRhoSoft.UtilityEditor.Icon

### Description

A helper class for creating a https://docs.unity3d.com/ScriptReference/Texture.html[Texture^] as a static field in editor classes. Icons can reference built in textures or loaded from https://docs.microsoft.com/en-us/dotnet/api/system.convert.frombase64string[base 64^] encoded data. This class is most commonly used with <<editor/label.html,Label>>.

### Static Methods

<<editor/icon.html,Icon>> BuiltIn(string _name_)::

Looks up a https://docs.unity3d.com/ScriptReference/Texture.html[Texture^] built in to Unity by _name_.

TIP: A list of built in icons can be found https://unitylist.com/p/5c3/Unity-editor-icons[here^]. The most common of these can be referenced using the static fields listed above.

<<editor/icon.html,Icon>> Base64(string _data_)::

Creates a https://docs.unity3d.com/ScriptReference/Texture.html[Texture^] from the base 64 encoded image _data_.

TIP: Many web services exist to convert files to a base 64 string. Search for 'convert to base64' to find one.

### Public Properties

https://docs.unity3d.com/ScriptReference/Texture.html[Texture^] _Content_ _(read only)_::

Retrieves the https://docs.unity3d.com/ScriptReference/Texture.html[Texture^] created for this Icon. The https://docs.unity3d.com/ScriptReference/Texture.html[Texture^] will be created or loaded on first access and reused after that.
